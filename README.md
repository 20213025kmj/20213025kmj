# 리눅스 명령어
------------------------
   ### 1. top
   > 실시간으로 CPU 사용률을 체크해주는 도구
  
   ### _top 정보 시스템 내용_
   * 00:00:00 현재 서버 시간
   * 0 days, 00:00 : uptime 켜져있는 시간
   * 0 users 유저의 수
   * load average 현재 시스템이 얼마나 일을 하고 있는지
   * Tasks 프로세스의 개수
     
   ### CPU
   * %us : 유저레벨에서 사용하고 있는 CPU 비중
   * %sy : 시스템 레벨에서 사용하고 있는 CPU 비중
   * %id : 유휴 상태의 CPU 비중
   * %wa : 시스템이 I/O 요청을 처리하지 못한 상태에서의 CPU idle 상태인 비중
     
   ### 메모리
   > Mem
   * total : 전체 물리적인 메모리
   * used : 사용중인 메모리
   * free : 사용되지 않는 여유 메모리
   * buffers : 버퍼된 메모리
   
   > Swap 
   * total : 전체 스왑 메모리
   * used : 사용중인 스왑 메모리
   * free : 남아있는 스왑 메모리
   * cached : 캐싱 메모리
